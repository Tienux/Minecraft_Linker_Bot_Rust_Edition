# Minecraft linker bot Rust Edition

A Discord bot that links your Minecraft server events to Discord.
Built in Rust using [Serenity](https://github.com/serenity-rs/serenity).


## Features

- 📢 Server start and stop announcements
- 📊 End-of-session stats report (deaths, blocks mined, playtime, and more)
- 💀 Real-time death announcements (PvP and PvE)
- 🟢 `/online` command — see who is currently connected
- 📈 `/stats [all/session]` command — see your session or all-time stats


- [Rust](https://www.rust-lang.org/tools/install) (stable)
- A Minecraft server
- A Discord bot token — [create one here](https://discord.com/developers/applications)
